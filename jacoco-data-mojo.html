graniteit:jacoco-data

Full name:

net.adamcin:graniteit-maven-plugin:0.7.0-SNAPSHOT:jacoco-data

Description:

Downloads the jacoco runtime data from the server using Sling JacocoServlet, which was introduced in org.apache.sling.junit.core:1.0.9-SNAPSHOT

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Since version: 0.8.0 .
  • Binds by default to the lifecycle phase: post-integration-test.
